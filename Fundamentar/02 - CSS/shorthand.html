Shorthand

É uma junção de propriedades

Exemplos

/*propriedades de fontes*/
font-style: italic;
font-weight: bold;
font-size: 8em;
line-height: 1.2;
font-family: Arial;

Considerações

  • Nesse caso não são consideradas propriedades anteriores, sempre será considerado a pós-propriedade
  • Para valores não especificados serão considerados os valores padrão
  • A ordem das propriedades não importa, podendo colocar então em qualquer ordem
