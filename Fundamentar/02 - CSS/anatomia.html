Anatomia do CSS

Vamos entender o que significa cada nome do CSS

  • Selector - é a tag selecionada do HTML para a mudança de estilo
    • Quando colocamos por exemplo o h1 para mudança e colocamos algumas propriedades, como mudança de cores e outros, essas mudanças irão afetar todos H1 em nossa página
  • Decoration - São todas as mudanças feitas no CSS para a mudança no HTML, como por exemplo, cor, tamanho da fonte, tipo da fonte, sombras e outros
  • Propeties - Para cada propriedade temos um valor da propriedade, as propriedades são por exemplo. color, font-size, backgound e outros
  • Property Value - São os valores atribuidos a cada propriedade, como por exemplo:
    • color: blue;
    • font-size: 50px;
