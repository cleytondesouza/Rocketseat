Adicionando CSS ao documento HTML

  • Estilo INLINE - Colocamos o código CSS dentro da própria TAG HTML
  • A segunda maneira de se fazer é pelo pelo head usando a tag style e colocando todas as configurações CSS entre essas tags
  • Podemos fazer também pela tag link rel="stylesheet" href="" que é colocada no head também.
  • Uma outra maneira é usando o @import, assim podemos importar fontes do fonts.google.com

É importante saber que a maneira mais utilizada é com tag link
