Parágrafo

Quando colocamos 2 ou mais comandos iguais para uma mesma tag, o documento vai aplicar as configurações do ultimo escrito

Porém se colocamos dentro da tag HTML o estilo para escrita, o documento obedecerá o que está dentro da tag em vez do que está entre as tags style ou no documento css linkado

Sendo assim a origem de onde nosso CSS está vindo importa muito, pois sabendo isso podemos fazer modificações rápidas e temporárias em nosso estilo

Podemos usar isso ha nosso favor quando precisamos forçar algum trecho a mudar de estilo caso seja necessário
