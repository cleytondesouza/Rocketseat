Comentários no CSS

Para comentar o CSS podemos usar o /* Conteudo do Comentário Aqui */

Os Comentários não vão afetar a nossa codificação

Eles nos ajudam a identificar algum trecho de código e ajuda também outros propgramadores

Ele pode nos ajudar a desabilitar um trecho do nosso código

Sempre que abrimos um comentário não podemos esquecer de fecha-lo
